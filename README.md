### Tecnologías y Herramientas
#### Desarrollo Backend & Frameworks
- **Python:** Especializado en frameworks rápidos como FastAPI, Flask y desarrollo/administración en entornos Odoo.
- **Java:** Creación de aplicaciones robustas utilizando Spring Boot.
- **Web & Frontend:** Construcción de sitios modernos y optimizados con Next.js y ecosistemas web actuales.

#### Herramientas & DevOps
- **Contenedores y Despliegue:** Docker para la virtualización y despliegue ágil de aplicaciones web.
- **Sistemas y Hardware:** Gestión de entornos de servidores y ensamblaje/optimización de hardware informático.

#### Diseño Gráfico y Emprendimiento
- Coordinación de diseño gráfico y gestión de logística/marketing para proyectos independientes y de personalización de indumentaria textil (Impresión DTF).
